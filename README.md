Projeto de classificação da qualidade do vinho usando Python.

## Conteúdo do projeto

- Análise exploratória dos dados.
- Criação da variável alvo binária.
- Pré-processamento com padronização.
- Treinamento de dois modelos: Regressão Logística e Random Forest.
- Validação com métricas: accuracy, precision, recall, F1 e ROC AUC.
- Geração de gráficos e exportação de resultados.
